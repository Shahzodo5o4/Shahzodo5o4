### 🏆 Featured project

#### ⚽ [Uzbekistan · World Cup 2026 — Prediction &amp; Tactical Analysis](https://github.com/Shahzodo5o4/uzbekistan-wc2026)

A statistical engine that learns from **49,000+ international matches (1872–2026)** to forecast
every game of the 2026 World Cup, simulate the whole tournament, and give tactical analysis for Uzbekistan.

- 📈 **Dixon–Coles (bivariate Poisson)** model — attack/defense strength + home advantage, validated with backtesting (log-loss 0.86 vs naïve 1.06)
- 🎲 **Monte-Carlo tournament simulation** (60,000 runs) → each team's title probability
- 🔴 **Live tracker** — predictions vs real results, auto-updating champion odds as matches are played
- 📊 **Interactive dashboards** — pure HTML/CSS/JS, no server needed

🌐 **Live demo:** https://shahzodo5o4.github.io/uzbekistan-wc2026/
